## Grid Column Details

### Complete Column Specifications

| Column | Width | Alignment | Sortable | Description |
|--------|-------|-----------|----------|-------------|
| Icon | 50px | Center | No | File type icon (dynamic) |
| Version | 80px | Center | Yes | Version number (1, 2, 3...) |
| File Name | Flex | Left | Yes | Original file name |
| Size | 100px | Right | Yes | Formatted size (KB/MB/GB) |
| Type | 100px | Center | Yes | File extension |
| Category | 150px | Left | Yes | Document type |
| Upload Date | 130px | Center | Yes | yyyy-MM-dd HH:mm |
| Uploaded By | 150px | Left | Yes | User name |
| Entity | 100px | Center | Yes | Parent entity (in project view) |

### Grid Sorting
- Default: Upload Date DESC (newest first)
- Click column header to sort
- Shift+click for multi-column sort

### Grid Filtering
- Search box: Filters by file name
- Entity Type dropdown: Activity, Risk, Meeting, Sprint, Project, User
- Document Type dropdown: All document type categories
- Clear button: Resets all filters

## Responsive Behavior

### Desktop (> 1200px)
- Full grid with all columns visible
- Sidebar with file details
- Upload dialog: Large (800x600)

### Tablet (768px - 1200px)
- Hide Entity and Category columns
- Reduced sidebar
- Upload dialog: Medium (600x500)

### Mobile (< 768px)
- Show only: Icon, File Name, Size
- Details in accordion below grid
- Upload dialog: Full screen

## Accessibility

### Keyboard Navigation
- Tab: Navigate between grid rows
- Enter: Open selected file actions menu
- Ctrl+U: Open upload dialog
- Ctrl+D: Download selected file
- Delete: Delete selected file (with confirmation)

### Screen Reader Support
- All icons have aria-labels
- Grid columns have proper headers
- File sizes read as formatted text
- Upload progress announced
- Success/error messages announced

## Performance Considerations

### Grid Loading
- Lazy loading: Load 50 items initially
- Virtual scrolling for large lists
- Pagination: 10/25/50/100 per page options

### File Upload
- Progress indicator with percentage
- Cancel button during upload
- Chunk upload for large files (> 10MB)
- Client-side validation before upload

### Download
- Stream large files (don't load in memory)
- Progress indicator for downloads
- Browser-native download dialog
