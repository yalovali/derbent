✅ How It Works
  • Color Assignment: Colors are generated from a hash of the user's name/initials for consistency
  • Initials: Extracted from first name and last name (e.g., "John Doe" → "JD")
  • SVG Format: Pure SVG with circle element for background and text element for initials
  • Size: 16x16 pixels (scalable without quality loss)
  • Rendering: SVG wrapped in div and appended as child element to work around shadow DOM
