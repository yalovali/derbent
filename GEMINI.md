## 🛡️ Non-Negotiable Enforcement
Gemini CLI strictly enforces all project-wide rules:
-   **C-Prefix Convention:** ALL custom classes MUST start with "C".
-   **Entity Constants:** ALL 5 mandatory constants (COLOR, ICON, TITLE_SINGULAR, TITLE_PLURAL, VIEW_NAME).
-   **Type Safety:** NO raw types; always use generics.
-   **Constructor Injection:** NO `@Autowired` fields.
-   **Initialization Patterns:** JPA vs Business constructor separation.
-   **Validation:** Service-layer `validateEntity()` mirroring DB constraints.

-   `analyze [pattern]` -> Assume Pattern Designer role.
-   `document [class]` -> Assume Documenter role.
